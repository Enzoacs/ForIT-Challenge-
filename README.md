Pasos para ejecutar mi Challenge ForIT:
1-Clonar el repositorio e introducirlo en su editor de codigo
2-Dirigirse hacia las carpetas
3-Instalar los modulos correspondientes (En Frontend y Backend)
4-Abrir una terminal para cada carpeta, asi el proyecto puede ser ejecutado sin errores
5-En terminal, ejecutar npm run dev (git)
6-Probar
